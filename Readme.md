# Bikash Kumar Yadav - Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, achievements, and projects as a software engineer. It is designed to provide an overview of my expertise in web development, competitive programming, and problem-solving.

---

## 🌟 Features

- **Responsive Design**: Fully responsive and optimized for all devices.
- **Skills Section**: Highlights my technical skills in frontend, backend, and problem-solving.
- **Projects Showcase**: Displays my key projects with links to live demos and GitHub repositories.
- **Contact Form**: Allows visitors to send messages directly to my email.
- **Interactive Animations**: Smooth animations using AOS (Animate On Scroll) library.


---

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript, jQuery, Tailwind CSS
- **Backend**: Node.js, Spring Boot (for some projects)
- **Libraries**: AOS (Animate On Scroll), Typed.js, Owl Carousel
- **APIs**: Web3Forms for contact form submissions
- **Hosting**: Deployed on [GitHub Pages](https://bikashkumaryadav.github.io/)
